### **Project Overview**

A modern React-based Library Catalog designed for academic and public browsing. This application leverages **TypeScript** for robust type safety and **Material UI (MUI)** to deliver a refined "Ivy League" aesthetic, focusing on visual hierarchy, accessibility, and high-performance data filtering.

### **Tech Stack**

* **Frontend:** React 18
* **Styling:** Material UI (MUI) v5
* **Language:** TypeScript
* **Deployment:** GitHub Pages

### **ğŸš€ Features**

* **Sophisticated "Ivy League" UI:** A custom-themed design featuring deep forest greens, vintage reds, and serif typography for an authentic library feel.
* **Advanced Multi-Criteria Filtering:** * **Range Search:** A visual rating filter allowing users to define minimum and maximum star ratings.
* **Category Tagging:** Instant toggle filtering using a "Chip" based selection system.


* **Global Search Omnibar:** Integrated title and author search with real-time feedback.
* **Accessibility & UX:** * Full keyboard navigation support (Enter/Tab) for interactive switches and buttons.
* Responsive design optimized for both mobile and desktop viewports.
* Sticky footer with automated copyright year and GitHub integration.


* **Favorites System:** Ability to toggle and isolate "Favorite" books with a single click.
